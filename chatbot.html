Chrome Built-in AI: Simple Chatbot

This example demonstrates a basic conversational chatbot using the `prompt()` method. It keeps a history of the conversation to provide context for the AI.

Hi! I'm a simple chatbot. How can I help you today?

A Note on Chrome Flags

To use this feature, you must have the "Enable built-in AI" flag enabled in Chrome. Open a new tab, paste the link below, press Enter, enable the flag, and then restart Chrome.
