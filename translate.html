Chrome Built-in AI: Simple Translator

This example demonstrates how to use the built-in AI to translate text from one language to another. It uses the `LanguageModel.translate()` method, which is the most direct and efficient way to perform this task.

A Note on Chrome Flags

While the `LanguageModel.translate()` API is stable in recent Chrome versions (138+), some platforms may still require the `chrome://flags/#enable-built-in-ai` flag to be enabled, especially on older versions or for other AI features.
